# IoT-Training

##### Pre-installation
- Install Node Version Manager or nvm
``` 
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash 
source ~/.bashrc
nvm 
```

- Install Nodejs version 10
``` 
nvm install 10.18.1
node -v
npm -v 
```

- Install Node-RED
``` 
npm install -g -unsafe-perm node-red 
```
